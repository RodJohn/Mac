# 获取镜像

    首先我们需要原版的安装镜像，
    这个直接去你的 APP Store 里面下载
    
    直接点击 macOS Sierra 最新版本下载。
    这样，你的应用程序里面会多出一个"安装macOS"的应用。


# UniBeast
 
    https://www.tonymacx86.com/resources/categories/tonymacx86-downloads.3/
           
           
注意，要使用下载 macOS 原版镜像的电脑制作，如果单单是把安装镜像拷贝到另外一台电脑上制作，由于两台电脑登陆的 Apple ID 不同，会导致安装U盘的制作失败。另外一点，Unibeast 软件打开前要记得将自己的电脑系统语言设置为英文，不然无法打开该软件。
